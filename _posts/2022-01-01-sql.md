## SQL 관련 메모

### Record vs Tuple vs Row

어디서 부르느냐에 따라서 차이가 있다.

| File System | Database Modeling | RDBMS       |
| ----------- | ----------------- | ----------- |
| File        | Entity            | Table       |
| Record      | Tuple             | Row         |
| Key         | Identifier        | Primary Key |
| Field       | Attribute         | Column      |

### Where vs Having

Having 은 Group By 뒤에 쓰인다. Group By 로 묶인 값들을 대상으로 Having 에 조건을 부여하는 것이다.
